# 📅 프로젝트 002: 서브컬처 게임 일정 대시보드 개발 (Subculture Game Schedule Dashboard)

본 프로젝트는 원신, 붕괴: 스타레일, 젠레스 존 제로, 명조, 명일방주 등 주요 서브컬처 게임들의 **이벤트 업데이트, 캐릭터 픽업, 공식 특별 방송, 점검 일정**을 한눈에 쉽고 예쁘게 볼 수 있도록 구축하는 **반응형 웹 대시보드 서비스**입니다.

이 대시보드는 깃허브(GitHub) 레포지토리에 소스 코드를 올리고 배포하여, 제작자 개인뿐만 아니라 다른 유저들과 시청자들도 언제든 편리하게 접속하여 일정을 확인하고 공유할 수 있도록 퍼블릭 서비스로 개발됩니다.

---

## 📌 1. 핵심 목적
* **시각적 편의성**: 여러 게임 사이트를 매번 확인하지 않고도, 서브컬처 팬들이 좋아하는 모든 게임 일정을 하나의 타임라인/달력으로 직관적이고 미려하게 조회합니다.
* **정적 사이트 배포**: 백엔드 서버 비용이 들지 않는 **GitHub Pages** 또는 **Vercel** 환경에 배포하여 평생 비용 0원으로 글로벌 오픈 서비스를 운영합니다.
* **자동 업데이트 융합**: 데이터 매니저 에이전트(003)가 수집하여 배포한 정적 JSON 데이터(`schedule_data.json`)를 클라이언트 단에서 fetch하여 UI에 실시간 렌더링합니다.

---

## 🛠️ 2. 기술 스택 및 화면 구성 구상

### ① 💻 기술 스택 (Frontend Stack)
* **Core Framework**: React (Vite로 경량화 빌드) 또는 Next.js (Static Export 모드)
* **Styling**: Vanilla CSS 및 TailwindCSS ( Harmonious한 서브컬처 감성의 다크 모드, 네온 글라스모피즘 스타일 디자인)
* **UI Components**: Lucide Icons (모던 이모지 아이콘), FullCalendar 또는 Custom Timeline Component
* **Deployment**: GitHub Actions 자동 빌드 ➔ GitHub Pages 배포

### ② 🖥️ 화면 구성 (UI/UX)
* **통합 타임라인 뷰**: 오늘을 기준으로 앞으로 예정된 모든 게임들의 패치와 방송 일정을 가로 타임라인 막대로 시각화 (예: `[명조 1.1 패치]` 막대와 `[젠존제 특별방송]` 막대를 오버랩하여 확인).
* **달력(Calendar) 뷰**: 월간 일정표 형태로 각 날짜별 픽업 시작일, 이벤트 마감일을 카드 형태로 정밀 표시.
* **게임별 필터 토글**: 픽업, 방송, 이벤트 카테고리별 / 원신, 스타레일, 명조 등 게임 브랜드별로 필터링하여 보고 싶은 정보만 핀포인트 조회.
* **픽업 디데이(D-Day) 타이머**: 최신 5성 캐릭터 픽업 종료 및 다음 픽업 오픈까지 남은 시간을 초단위 카운트다운 타이머로 렌더링.
